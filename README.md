## 介绍

本项目旨在通过上位机、手套和视觉系统对机械手进行全面控制。每个部分都有独立的模块和详细的说明，确保用户可以轻松上手。

## 上位机

上位机部分的代码用于与机械手进行通信和控制。它提供了一个图形用户界面（GUI），用户可以通过该界面发送指令、接收反馈并监控机械手的状态。

### 功能

- 发送控制指令
- 接收并显示反馈
- 实时监控机械手状态

### 依赖

- Python 3.x
- PyQt5
- PySerial

### 使用

```bash
cd host_computer
pip install -r requirements.txt
